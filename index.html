Wanted

Telephony API Ideas !

XOXZO is seeking new ideas for our telephony API applications
and hosting XOXZO Code PyCon APAC 2017, an open source code developing contest.

Winner gets $100 Amazon gift credit !

Contest Rules

  • You must use XOXZO phone API (simple playback, tts playback, DIN). SMS api is not counted. for details about how to use, please look at this page.
  • Submit your project on public open source repository on github, gitlab, bitbucket, etc. and send its url to codecontest@xoxzo.com via email.
  • Submission must contain source code, tests and README documents on how to install or use
  • Submission must be open source licensed.
  • Participant must sign up XOXZO account first. Please visit here to sign up.
  • Participant is limited to attendee of PyCon APAC 2017
  • Implementation language is not limited to python.
  • If you have any questions, please contact codecontest@xoxzo.com via email.

Schedule

  • Code submission dead line: Sep. 3 (Sun) 2017
  • Winner announcement: Sep. 11 (Mon) 2017

Prize

Winner awarded $100 Amazon gift card for five people/teams.

Judging Criteria


Based on the following judging criteria, XOXZO Code PyCon committee will chose the winner.
  • Uniqueness/Ingenuity
  • Feasibility
  • How many potential users the plugin/app will have? (i.e if it's a plugin for a very famous CMS)

Sample App category(but not limited to these)

  • Web app
  • Mobile app
  • CMS plugin
  • Bot plugin
  • Python package.
